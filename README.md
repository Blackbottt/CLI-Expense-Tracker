# CLI Expense Tracker

A simple command-line application to track personal and business expenses, built with Python.

This project focuses on input validation, data persistence, and CLI-based user interaction.

## Features

- Add, edit, and delete expenses
- Categorize expenses (business, personal, household)
- Filter by category
- Monthly summary
- Persistent storage using JSON

## Tech Stack

- Python
- JSON (for storage)
- CLI (Command Line Interface)
